# Omnivox Scraper

Calcule la moyenne générale d'un compte étudiant Omnivox.

## Installation

```bash
git clone <LIEN_REPO>
npm install
```

Ensuite, créez un fichier .env et ajoutez-y les variables suivantes:

```
OMNIVOX_LOGIN_LINK = "link_to_login_page"
```

## Utilisation

```bash
npm start
```
